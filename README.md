# CuHotbar V1.0.2 for Casualties: Unknown

> 适配游戏《未知伤亡》的快捷物品栏模组，仿 Minecraft 物品栏
> 兼容：SaveManager（在场时设置合并到其面板，缺少不影响功能）

English guide: see [README.en.md](README.en.md)

## 简介

该模组在游戏内提供一条可自定义的快捷物品栏：从原版背包拖动物品到槽位建立绑定，或鼠标指向物品按对应槽位键直接绑定；用各槽自定义的按键或滚轮快速把物品切到主手，并能用自定义按键（默认 Shift+鼠标右键）使用手中物品。槽位数量不限，支持横排（默认靠下）/ 竖排（默认靠右）并可分多排多列显示，每个槽位的切换键都能单独绑定（支持组合键），槽位左上角实时显示其绑定的按键。快捷栏位置、缩放、透明度均可调。绑定按角色独立缓存，重进游戏不丢失。检测到 SaveManager 时自动把设置合并为其面板的一个分页，缺失时则独立注入按钮与面板。

## 功能

- **拖拽绑定**：从身上 / 背包拖动物品到槽位即可绑定；一个物品只占一个槽，重复拖入会自动从旧槽移除。
- **指向绑定**：鼠标指向背包格或地面物品时，按某个槽位的切换键即可绑定，无需拖拽。
- **每槽自定义键**：每个槽位的切换键都能单独绑定，支持组合键（如 Ctrl+1）；前 9 槽默认小键盘 1-9，新增槽位默认未绑、可自行设置。
- **按键角标**：槽位左上角显示该槽实际绑定的按键，改键后即时更新；未绑定则留空。
- **快速切换**：按各槽绑定键或滚动鼠标滚轮切换当前槽并把物品切到主手；选中槽有白色加粗边框。
- **自定义使用键**：用可配置的按键使用手中物品，支持组合键，默认 Shift+鼠标右键（可在设置改键或清空关闭）。
- **快捷安全使用**：可选开关。开启后按槽位键或滚轮选中食物 / 饮品 / 口服药等可食可饮的物品时不切到主手，主手原物品保留；按使用键即可直接吃 / 喝（按物品实际行为识别，工具 / 炸药 / 弹匣等不会被误用）。
- **横竖排与多排**：槽位数量不限，可选横排（默认靠下）或竖排（默认靠右），并设置排数 / 列数按网格分布。
- **安全归还**：切换 / 收起物品时，上一件按「原位置 → 身上空位 → 掉落」的顺序放回；切换时主手物品无空槽则优先塞回背包容器，确实放不下才掉落并提示。
- **自动补充**：手中物品丢出或用尽后，若身上仍有同种则自动补到主手（可在设置关闭）。
- **状态显示**：槽位显示身上同种物品数量，并以底部状态条表示电量 / 弹匣余弹 / 耐久。
- **外观可调**：排列方向、槽位数量、排数 / 列数、整体缩放、背景透明度（不影响物品图标）、屏幕锚点与偏移均可设置。
- **按角色缓存**：快捷栏绑定按角色独立保存，重进游戏自动恢复。

## 使用方法

### 安装

把 `CuHotbar` 文件夹放到 `BepInEx/plugins/` 下，内含：

- `CuHotbar.dll`

启动游戏后，主菜单会注入「CuHotbar」按钮；游戏内按 ESC 打开暂停面板时右下角也有唤起按钮。检测到 SaveManager 时，设置改为合并进其面板的「CuHotbar」分页。

### 基本操作

| 操作 | 默认按键 / 方式 |
|------|----------------|
| 绑定物品到槽位 | 拖物品到槽位，或鼠标指向物品按对应槽位键 |
| 切换并切到主手 | 各槽绑定键（前 9 槽默认小键盘 1-9）/ 鼠标滚轮 |
| 使用手持物品 | Shift+鼠标右键（可改键） |
| 从槽位取出 | 从槽位拖出（拖到肢体用药、拖回背包等沿用原版逻辑） |

### 设置面板

| 分区 | 用途 |
|------|------|
| 显示 | 显示开关、槽位数量、排数 / 列数、排列方向（横排 / 竖排）、整体缩放、背景透明度 |
| 切换按键 | 每个槽位的切换键、使用主手物品的按键、滚轮切换、自动补充、掉落提示、快捷安全使用 |
| 位置 | 屏幕锚点 X/Y、像素偏移 X/Y、恢复默认位置 |
| 快捷键 | 打开 / 关闭设置面板 |
| 其他 | 在游戏控制台显示模组日志、接受新版本更新提示 |

> 切换键、使用键均支持组合键。点击对应按钮后按下新按键（含鼠标键）即可绑定，按「清除」取消绑定。指向物品按槽位键即绑定该物品、未指向时则为切换。横排默认靠屏幕下方、竖排默认靠右；切换方向会套用对应默认位置。槽位数量与排数可一直增加，设过大时槽位可能超出屏幕。

## 软依赖说明

- **SaveManager**：在场时本模组通过其扩展点把设置注册为一个分页，风格统一；缺失或版本无扩展点时自动回退为独立的主菜单按钮 + 游戏内浮动按钮 + 独立设置窗口。

## 更新检测

启动时比对 GitHub 最新版本，有新版在屏幕左上角红字提示，点击打开 release 页；可在「设置 → 其他」关闭。
