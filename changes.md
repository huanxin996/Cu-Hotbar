# CuHotbar 更新日志

> English changelog: see [changes.en.md](changes.en.md)

## v1.0.2

### 新增

- **横排 / 竖排切换**：可选择快捷栏排列方向。横排默认靠屏幕下方，竖排默认靠右侧；切换方向时自动套用对应的默认位置。
- **多排 / 多列**：横排时“排数”为行数、竖排时为列数，槽位按网格分布。
- **指向物品绑定**：鼠标指向背包格或地面物品时，按某个槽位的切换键即可把该物品绑定到该槽，无需拖拽。
- **快捷安全使用**：可选开关。开启后按槽位键或滚轮切换到可食用 / 饮用的消耗品（按物品的实际使用行为识别——是否进食 / 饮用，覆盖地心果等分类为 custom 的食物，含有副作用的增益药也算）时不切到主手，保持主手物品不动；是否使用由你设的「使用键」（默认 Shift+右键）决定。使用键按选中槽类型分流：选中消耗品时使用其绑定物品（可在背包），选中其他物品时只使用主手里的那件——炸药、弹匣、注射器等不会被识别为消耗品，也不会被使用键误触发。

### 变更

- 切换物品时若主手物品无处安放，不再直接掉落：优先放回身上空槽，其次塞入可容纳的背包容器；确实放不下才掉落，并在屏幕给出提示（提示可在设置中关闭）。

## v1.0.1

### 新增

- **每个槽位可单独绑定切换键**：不再是写死的小键盘 1-9，每个槽都能单独设键，支持组合键（如 `Ctrl+1`）。前 9 个槽默认仍是小键盘 1-9，新增的槽位默认未绑、可自行设置。
- **槽位角标显示实际绑定键**：槽位左上角的角标改为显示该槽真正绑定的按键，在设置里改键后即时更新；未绑定则留空。
- **使用主手物品支持自定义组合键**：原“右键使用”开关升级为可配置按键，支持组合键，默认 Shift+鼠标右键；清空该绑定即可关闭此功能。
- **多排布局**：新增“排数”设置，槽位可按网格分多排显示。
- **槽位数量解除上限**：槽位数量不再限制在 9 个，可按需增加。

### 变更

- 设置面板“切换按键”分区重做：移除“数字键基准（小键盘 / 主键盘）”与“右键使用”开关，改为逐槽按键 + 使用键的绑定行；槽位数量、排数改为 -/+ 步进调节。

### 说明

- 切换键与使用键均支持鼠标键与组合键；在设置面板点击对应按钮后按下新按键即可绑定，点“清除”取消绑定。
- 槽位数量与排数可持续增加，设得过大时槽位可能超出屏幕，请配合排数与缩放调整。
